# ❌⭕ Tic-Tac-Toe

A classic Tic-Tac-Toe game for two players.

## 🎮 How to Play
- First player to click becomes ❌
- Second player becomes ⭕
- Take turns clicking on the board
- First to get 3 in a row wins!

## ✨ Features
- Two player mode
- Win detection logic
- Responsive design for mobile

## 🛠 Technologies
- HTML
- CSS
- JavaScript
